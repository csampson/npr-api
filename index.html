Public Radio Tuner API

This a REST api for querying information about NPR member stations. It features filtering, geolocation searching, pagination, and sorting.

Default response content-types: application/json
Schemes: http

Summary

Path Operation Description
/stations GET

Fetch a collection of NPR member station records

Paths

Fetch a collection of NPR member station records

GET /stations

When needed, pagination information will be encoded for you in the Links header in the response, e.g:

<http://api.publicradiotuner.io/stations?page=3>; rel="next", <http://api.publicradiotuner.io/stations?page=29>; rel="last", <http://api.publicradiotuner.io/stations?page=1>; rel="prev"
filter

Filters the list of stations using key-value pairs of station attributes

 query string (json)
sort

Sorts the list of stations by a given station attribute value

 query string
page

Page number to show stations from (page size is 30 items)

 query number (int32)

Uses default content-types: application/json

200 OK

An array of NPR member stations
